Starmap Hotkeys

Double Click Star Icon = Open UESP Wiki Link
Alt-Click Star Icon = Show System Properties
"r" over Star Icon = Show Resource Wheel
"h" = Show Infobase Links
"j" = Show Global Journal
"s" = Show System Journal
"c" = Show Codex Journal
"F3 or CTRL+F" = Open Search Bar
"F5" = Refresh Page
"F12" = Browser Debug Mode

Star Icon States

Cyan Font = Discovered
Green Font = Surveyed(100%)
Outpost Icon = Player Outpost

Marker Icons

= Major Settled Location
= Ship Build Location
= Player Outpost

Radio Widget

To activate the Radio Widget, please click on the Starfield icon on the right side of the "Starfield Map Helper" banner.

For further instructions on the Radio Widget, please click on the "Starfield Radio" label of the widget banner. Extended instructions will be added in future versions.
